# Sales Data Analysis Dashboard (Excel)

## Project Overview
This project analyzes sales data using Microsoft Excel and presents insights through an interactive dashboard.

## Key Features
- Data cleaning and formatting
- Pivot tables for sales analysis
- Interactive charts (Line, Column, Pie)
- Slicers for dynamic filtering

## Tools Used
- Microsoft Excel

## Insights
- Electronics generated higher revenue than Accessories
- Male customers contributed more to total sales
- Cash and UPI were the most used payment modes
